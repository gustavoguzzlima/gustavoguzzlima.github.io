GUSTAVO D. J. LIMA
// Full-stack Developer

/* Resumo */

Meu objetivo profissional é aplicar conhecimentos de computação no projeto e implementação de soluções eficientes, robustas e escaláveis.

Depois que tive contato com desenvolvimento front-end, em 2018, aprendi tecnologias como React e Redux, Node e versionamento com Git. Em seguida resolvi estudar Vue, Angular e Git Flow, e em 2020 comecei a atuar como desenvolvedor web e trabalhei na implementação de alguns temas para WordPress. Após certo período, comecei a trabalhar no setor de telecomunicações como programador, com o objetivo de identificar problemas, modelar soluções e desenvolver produtos.

Atualmente atuo como desenvolvedor full-stack, onde desenvolvo integrações entre sistemas, APIs REST com OAS, automação e tratamento de dados com Python e trabalho no desenvolvimento de soluções internas em Angular e Express. Ao longo do tempo adquiri experiência com uso e administração de sistemas Linux/UNIX e noções em tópicos como virtualização (Proxmox), cloud computing (GCP) e gerenciamento de projetos de software com Agile (Scrum e Kanban).
